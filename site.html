📋 О курсе Python для начинающих

Цель курса — дать системное введение в программирование на Python с нуля.

  • Понимать синтаксис и базовые конструкции языка.
  • Работать с переменными, условиями, циклами, функциями, структурами данных.
  • Читать и записывать файлы, обрабатывать ошибки, импортировать модули.
  • Владеть основами объектно-ориентированного программирования.
  • Писать собственные программы средней сложности.
  • Продолжить обучение в веб-разработке, анализе данных, автоматизации.

Для кого

Для абсолютных новичков.

Структура

  • 8 лекций в PDF.
  • 10 лабораторных работ (включая две проектные).
  • Итоговый тест из 35 вопросов.

Как пользоваться

Лекции и лабораторные открываются на странице. Используйте кнопку «На весь экран» для увеличения.
